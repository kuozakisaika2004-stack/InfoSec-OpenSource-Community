# 数据投毒攻击模拟实验

## 功能概述
本实验基于 PyTorch 深度学习框架，在 MNIST 手写数字识别数据集上训练了一个简化版 AlexNet 卷积神经网络。实验模拟了 **数据投毒攻击** 场景：将部分训练样本的标签篡改为随机错误值，观察投毒对模型分类准确率的影响。同时，代码提供了训练过程可视化、分类结果展示等功能，便于分析攻击效果。

## 代码文件说明
- `poison_alexnet_mnist.py`：主程序文件，包含以下核心模块。

## 代码结构与核心功能

### 1. 神经网络模型定义（`AlexNet` 类）
- **功能**：构建一个适用于 MNIST（28×28 灰度图）的简化 AlexNet 结构。
- **细节**：
  - 5 个卷积层（Conv2d）+ 3 个最大池化层（MaxPool2d）
  - 3 个全连接层（Linear），输出 10 个类别
  - 输入通道数为 1（灰度图），输出层为 10 分类

### 2. 数据集处理与投毒策略（`fetch_datasets` 函数）
- **功能**：将原始训练集按指定比例（本实验为 50% 投毒 + 50% 干净）划分。
- **投毒方式**：对投毒样本随机替换标签为 0~9 中的任意数字（与原始标签可能不同）。
- **返回结果**：分别得到 `poison_trainset`（投毒样本集）和 `clean_trainset`（干净样本集），最后合并为完整训练集。

### 3. 训练过程与评估
- **损失函数**：交叉熵损失（`CrossEntropyLoss`）
- **优化器**：Adam 优化器（学习率 0.01）
- **训练轮数**：20 个 epoch
- **评估指标**：每个 epoch 结束后，在测试集上计算分类准确率，并记录平均损失值。

### 4. 可视化函数
| 函数名 | 功能描述 |
|--------|----------|
| `plot_correctly_classified_images()` | 展示模型预测正确的样本图像，标题显示真实标签与预测标签 |
| `plot_misclassified_images()` | 展示模型预测错误的样本图像，便于分析模型易错类别 |
| 训练曲线绘制（末尾部分） | 分别绘制准确率随 epoch 变化的折线图，以及损失值下降曲线 |

### 5. 日志记录
- 训练过程中的每个 epoch 的损失值会同时输出到控制台和 `training_log.txt` 文件中，便于后续查看与分析。
